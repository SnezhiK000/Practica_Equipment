# Система учёта заявок на ремонт техники

Веб-приложение для автоматизации процесса учёта и обработки заявок на ремонт техники в ГБУЗ РХ «Абаканская межрайонная клиническая больница».

## Описание

Система предназначена для:
- Регистрации заявок на ремонт оборудования сотрудниками организации
- Назначения исполнителей и контроля выполнения работ
- Учёта оборудования и его местоположения
- Расчёта затрат на ремонт и обслуживание техники
- Формирования статистики и отчётности

##Технологии

- **Backend**: Django 6.0.3 (Python 3.13)
- **Database**: PostgreSQL 14+
- **Frontend**: HTML5, CSS3, Django Templates
- **Architecture**: Клиент-серверная, 3 уровня

## Установка

### Требования
- Python 3.10+
- PostgreSQL 14+
- pip и virtualenv

### Пошаговая установка

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/SnezhiK000/Practica_Equipment.git
cd equipment_p/equipment
```

2. **Создайте и активируйте виртуальное окружение**
```bash
   python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux/Mac
source .venv/bin/activate
```
3. **Установите зависимости**
```bash
pip install -r requirements.txt
```
4. **Настройте базу данных**
```bash
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'equipment_db',
        'USER': 'postgres',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '5432',
        'OPTIONS': {
            'options': '-c search_path=equipment'
        },
    }
}
```
5. **Примените миграции**
```bash
python manage.py makemigrations
python manage.py migrate
```
6. **Запустите сервер разработки**
```bash
python manage.py runserver
```
7. **Откройте в браузере**
```bash
[python manage.py runserver](http://127.0.0.1:8000/)
```
